# vim-fundamentals

**vim-fundamentals** introduces the Vim text editor through **Normal** and **Insert** modes only.
The repository is designed for beginners, classroom teaching, and self-paced learning.

## Learning Objectives
- Understand Vim’s modal editing model
- Navigate and edit files confidently
- Insert, cut, copy, paste, and move text
- Search and replace content
- Use yanking and marking
- Recover safely using undo and escape commands

## Repository Structure
```
vim-fundamentals/
├── README.md
├── lessons/
│   └── vim-normal-insert.md
└── exercises/
    └── demo.txt
```

## How to Use
1. Read `lessons/vim-normal-insert.md`
2. Open the practice file:
   ```bash
   vim exercises/demo.txt
   ```
3. Follow the lesson and practice each command.

## Scope
- Normal and Insert modes only
- Focus on core Vim operations
- Suitable for Linux, DevOps, and SSH environments
